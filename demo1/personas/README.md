# 모의 상담 페르소나 인덱스

IBK 기업은행 "AX 실습 — 모의 상담"용 모의 고객 페르소나. `data/`의 실제 상담 내역을 기반으로, **주제 3종 × 난이도 3단계(하·중·상) = 총 9개**로 구성했다. 각 파일은 agent에 개별 주입해 **고객 역할 연기**에 사용한다.

## 사용법
- agent에 페르소나 파일 1개를 시스템/컨텍스트로 주입 → agent는 해당 고객을, 사용자(교육생)는 상담 직원을 연기
- 각 파일의 `연기 지침`이 톤·저항 포인트·정보 공개 방식·반응 패턴을 규정
- 교육 단계에 맞춰 난이도(하→중→상)를 선택

## 난이도 기준 (3주제 공통)
- **하**: 협조적·단순 요구, 표준 절차/상품으로 해결, 감정 동요 적음
- **중**: 복합 니즈나 오해·항의가 있어 추가 설명·대안 제시 필요
- **상**: 상황이 복잡(다수 이해관계·불확실성)하거나 감정이 격앙되어 정교한 위기·예외 응대 필요
