# IBK 기업은행 AX 실습 — 모의 상담 에이전트

가상의 상담 내역을 기반으로 만든 **모의 고객**과 상담을 연습하는 AI 에이전트입니다.
에이전트가 **고객(상담 요청자)** 을 연기하고, 사용자가 **IBK기업은행 상담 직원**이 되어 응대합니다.
고객이 말할 때마다 **`(추천 답변: ...)`** 이 함께 표시되어 학습을 돕습니다.

---

## 빠른 시작

이 프로젝트를 연 에이전트(Codex 등, `AGENTS.md`를 읽는 도구)에게 이렇게 입력하세요.

```
상담 연습 시작해줘
```

그러면 에이전트가 순서대로 물어봅니다.

1. **상담 종류** — `기업대출` · `예적금` · `민원/사고`
2. **난이도** — `하` · `중` · `상`

선택이 끝나면 해당 페르소나를 불러와 고객 역할로 상담을 시작합니다.

---

## 진행 방식

- 에이전트는 **고객**으로 말하고, 각 발화 다음 줄에 상담 직원이 참고할 **추천 답변**을 붙여줍니다.
- 사용자는 상담 직원으로서 자유롭게 응대하면 됩니다 (추천 답변을 그대로 써도, 자기 표현으로 바꿔도 됩니다).
- 고객은 정보를 한꺼번에 말하지 않고, 직원이 물어보는 것에 답하며 오해·저항을 자연스럽게 드러냅니다.

### 대화 예시 (예적금 · 중)

```
고객: 제가 3년 부었던 적금이 이번 달에 만기가 됐는데, 이 돈을 어떻게 할지 모르겠어요.
(추천 답변: 만기 축하드립니다. 이미 목돈이 되셨으니 매달 나눠 넣는 적금보다
          한 번에 예치하는 정기예금이 이자 면에서 유리합니다.)

직원: (사용자가 응대)

고객: 왜요? 적금 금리가 더 높던데요.
(추천 답변: 표면 금리는 적금이 높아 보이지만, 적금은 매달 넣는 돈마다 이자 붙는 기간이
          달라 실제 수령 이자는 표면금리의 절반 수준입니다. 목돈은 정기예금이 유리해요.)
```

### 진행 중 명령어

| 입력 | 동작 |
|------|------|
| `[힌트 줘]`, `[잠깐 멈춰]` | 대괄호 `[ ]` 지시 → 잠시 캐릭터에서 나와 도움을 준 뒤 복귀 |
| `상담 종료` / `끝` / `그만` | 상담을 마치고 **코칭 피드백** 제공 (잘한 점 · 놓친 부분 · 개선 제안) |

---

## 상담 시나리오 (페르소나)

주제 3종 × 난이도 3단계 = 총 9개. 모두 `data/`의 실제 상담 내역에 근거합니다.

### 기업대출
| 난이도 | 고객 | 상황 |
|--------|------|------|
| 하 | 박대성 (대성정밀공업) | 운전자금 · 원자재 대금 결제 |
| 중 | 정루아 (루아패션) | 소상공인 정책자금 · 재고 사입 |
| 상 | 조대명 (대명건설자재) | 부동산 담보대출 · 거래처 부도 긴급자금 |

### 예적금
| 난이도 | 고객 | 상황 |
|--------|------|------|
| 하 | 사회초년생 | 주택청약종합저축 신규 가입 |
| 중 | 알뜰 주부 | 적금 만기 재예치 (예금 vs 적금) |
| 상 | 정년퇴직자 | 퇴직금 3억 자산 분배 · 절세·예금자보호 |

### 민원/사고
| 난이도 | 고객 | 상황 |
|--------|------|------|
| 하 | 정수빈 | ATM 출금 오류 |
| 중 | 강민철 | 대출 금리·중도상환수수료 이의 |
| 상 | 김서연 | 보이스피싱 (검찰 사칭) |

> 난이도 기준 — **하**: 협조적·단순 / **중**: 오해·항의·복합 니즈 / **상**: 복잡·감정 격앙(위기 응대)
